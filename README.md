# Qiaopi Archive Intelligence

侨批智档是面向侨批数字保护的公益平台展示网站，服务侨批持有人、侨胞后代、学校师生、公共文化机构、研究者和侨乡村庄。

网站围绕侨批从原件保存到公共文化服务的完整路径，呈现以下内容：

- 侨批图像采集、增强、病害标注与数字建档
- 面向手写繁体、异体字和竖排文本的 OCR/HTR 识别与人工校对
- 人物、地点、时间、金额、亲属关系和迁徙线索的语义结构化抽取
- 区分演示证据与未检索模型回答的可信问答界面
- 授权分级、隐私脱敏、审核状态和公开边界管理
- 寻根辅助、教育展陈、研究检索与侨乡公共文化服务

网站以东溪村为首期试点，展示可复核、可迁移、可复制的侨批数字保护工具与公益服务方案。
